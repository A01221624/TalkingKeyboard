MultikeyTextModel
This class contains the model of currently held multi-character text. It is used by the suggestion service module to disambiguate between the different possible words which can be formed by using the previously selected characters and to suggest auto-completion candidates. The current version of the application holds an alternative model directly in the suggestion service; both approaches have benefits and thus non-operational class is included.
