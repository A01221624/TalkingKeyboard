TalkingKeyboard.Infrastructure.DataContainers Namespace
The Talkingkeyboard.Infrastructure.DataContainers namespace provides class models for holding and processing data which may be used by several modules.
Classes
  Class Description
MaintainablePointCollectionT1, T2
Defines the MaintainablePointCollection abstract class used as a base for collections of points mapped to a given object (e.g. control, time).
TimedControlsWithPoint
Defines the TimedControlsWithPoint class which maintains a collection of points (each with the time at which it was obtained) mapped to the control they fall on.
TimedPoints
Defines the TimedPoints class which maintains a collection of points along with the time they were obtained.
TimePointPair
This class is used to hold pairs of Point-DateTime.
Interfaces
  Interface Description
IMaintainable
Defines the IMaintainable interface used to make collections maintainable (e.g. eliminating old elements).
