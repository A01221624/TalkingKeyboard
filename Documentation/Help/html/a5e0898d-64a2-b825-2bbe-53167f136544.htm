MicrosoftTextToSpeechModule
Class for initializing the module for providing text to speech services, particularly obtained by the Microsoft TTS (which uses the voices installed for Windows).
TextToSpeechService
This class implements the service for providing text-to-speech capabilities. The internal engine is currently the one provided with Windows systems. The current implementation reads directly the entire text from the model.
